A state-of-the-art, beautifully designed recipe dashboard that brings diverse, gourmet culinary experiences straight to your screen. From meticulous search capabilities to an immersive glassmorphic UI, this dashboard makes browsing world-class recipes delightful.

**Live Demo:** [Check out the live app here!](https://bro-recipes.vercel.app/)

---

## ✨ Features

- **Premium Dark Mode UI**: Boasts a modern, high-contrast dark theme powered by rich, violet-magenta gradient accents.
- **Glassmorphic Design**: Utilizes ultra-modern frosted glass layers, blurred backdrops, and soft ambient shadows.
- **Real-Time Search**: Seamlessly filter through dozens of global recipes by title, cuisine style, or ingredients instantly.
- **Micro-Animations & Transitions**: Enjoy silky smooth hover effects, accordion reveals, layout fade-ins, and a custom interactive loading state.
- **Responsive & Accessible**: Fully adaptable to any form-factor, leveraging modern typography (Outfit & Inter fonts).
- **Rich Iconography**: Integration with `lucide-react` to provide sharp, highly readable UI feedback (star ratings, timer spans, difficulty caps).

## 🛠️ Technology Stack

- **Core**: React 18, Vite (Fast build tooling)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid, Viewport optimizations, custom keyframe animations)
- **Icons**: Lucide React
- **Data Source**: Fetching dynamic recipe data from [DummyJSON](https://dummyjson.com/recipes)
- **Deployment**: Vercel

## 🚀 Getting Started

If you'd like to run the project locally, just follow these simple setup steps:

### Prerequisites

- Node.js (v16.x or later)
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yuvaraj007A/bro_recipes.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd bro_recipes
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Spin up the development server:**
   ```bash
   npm run dev
   ```
   
5. **Open your browser:** Visit `http://localhost:5173` to see your Gourmet Dashboard running!
